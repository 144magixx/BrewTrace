# 前端说明

前端现在是 React + Vite + TypeScript 本地工作台。首页直接进入“当前记录”工作流，不是 landing page。

## 安装依赖

```bash
npm install
```

## 常用命令

```bash
npm test
npm run build
npm run test:e2e
npm run dev
```

开发服务启动后监听：

```text
http://127.0.0.1:5173
```

Vite 已配置 `/api` 代理到 `http://127.0.0.1:8080`，因此本地验收时先启动后端，再启动前端。

## 验收入口

1. 点击“开始记录”创建会话。
2. 输入“今天喝了一支水洗埃塞，有柑橘和红茶感”。
3. 确认页面追问豆子、冲煮参数和文案风格，且不生成最终草稿。
4. 继续输入“豆子是某烘焙商的埃塞水洗豆，水温 92 度，粉水比 1:15，想看克制、夸张和锐评。”。
5. 确认克制版、夸张版和锐评版草稿出现，并显示事实边界说明。
