## 风格定位

这是“克制、完整、可复盘”的咖啡记录风格。它不是没有观点，而是不靠情绪推进观点。判断必须来自豆子信息、冲煮参数和杯中表现；可以夸好，也可以指出缺点，但语气保持平稳。

中立风格由两个子风格组成：

- 斯慕吉Smoothie：完整档案型，像把每支豆子归档。
- Apple：温度分段型，像口腔感手账。

## 斯慕吉Smoothie：完整档案型

斯慕吉正文高度结构化，常见顺序是：

1. 豆子：国家、庄园、地块、品种、处理法、烘焙商。
2. 器具：滤杯、滤纸、磨豆机、壶。
3. 参数：研磨刻度、粉量、粉水比、水温。
4. 冲煮：预浸、分段注水、结束时间、bypass。
5. 评价：香气、甜感、酸质、干净度、燥感、养豆、火候、性价比。

它的观点不弱，但每个观点都有依据。常见判断方式是“干净度非常高”“燥感明显”“火候不足”“甜感支撑好”“性价比遗珠”。正文通常较长，读感像咖啡豆档案和冲煮实验记录。

## Apple：温度分段型

Apple 的正文更碎片、更口语，常用中英混合记录豆名、批次和烘焙商，再按“开袋 / 高温 / 中温 / 凉下来 / 参数”推进。它会保留真实细节，例如卡片信息、短斤缺两、兑水 10ml、一下喝完、这包和新鲜时差异不大。

它的重点不是完整百科，而是温度变化中的口腔感：热的时候是什么香，温降后出现什么果皮、茶感、酸质、甜感，最后再给一个克制结论。

## 语言特征

- 标题准确，不夸张，常写产区、庄园、处理法、品种、烘焙商或风味概括。
- 正文常出现“豆子：”“器具：”“参数：”“冲煮：”等栏目。
- 判断带限定词：这次、这一版、温度下来后、个人觉得、整体看。
- 可以指出不足：燥感、干净度一般、尾段空、甜感支撑不足、温降后变薄、火候略重。
- 不攻击人，不用锐评口吻，不写广告口播。

## 写作提示词

请生成 restrained / 中立克制风格的小红书咖啡品鉴：

1. 标题准确写出产区、庄园、处理法、品种或烘焙商，不用夸张词。
2. 正文先交代基础信息，再写参数，最后写风味变化和结论。
3. 所有评价都要绑定杯中依据，不要空泛说“神”“封神”“吊打”。
4. 使用“这次喝到”“这一版表现为”“温度下来后”限定判断。
5. 缺点可以写清楚，但语气保持中立。
6. 需要严谨时用斯慕吉结构；需要口腔感碎片时用 Apple 结构。
7. 标签控制在 2-5 个，优先咖啡品鉴、手冲咖啡、我的咖啡日记。
