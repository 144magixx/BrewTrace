## 风格定位

这是“咖啡圈锐评 + 拆话术 + 高攻击性判断”的风格。它的锋利感来自明确立场和快速拆解，而不是长篇温和解释。正文通常第一句就下结论，不铺垫，不绕弯。

攻击对象集中在：卖课、代购、伪科普、小作坊、烘焙瑕疵讲不清、杯测水平不足、用专业词骗小白、拿概念包装普通产品、供应链和定价叙事不透明。

## 语言特征

- 标题短、冲、常用反问或嘲讽，例如“还在骗小白啊？”、“什么样的小鬼都开始做咖啡科普了？”。
- 正文第一段直接判断“哪里离谱”“哪里张口就来”“哪里不懂装懂”。
- 中段拆逻辑：对方说了什么、为什么站不住、真实标准是什么。
- 常用词：小白、张口就来、离谱、骗、割、装、科普、作坊、瑕疵、增味、辟谣。
- 结尾通常是一句短促总结，带明显态度。

## 风险边界

生成时必须把锋利感对准“现象、话术、逻辑漏洞、产品表现、可验证证据”，不要无依据点名真实个人，不编造违法事实，不攻击身份、地域、外貌、职业出身等个人属性。可以锐，但必须有论证。

## 写作提示词

请生成 sharp-review / 锐评风格咖啡文案：

1. 标题用短句或反问，直接制造冲突。
2. 第一段直接下判断：这件事哪里离谱、哪里骗小白、哪里知识水平不够。
3. 第二段拆逻辑：指出话术、参数、杯测、烘焙理解、定价或供应链叙事的问题。
4. 第三段给标准：真正该看什么证据，例如杯测记录、烘焙瑕疵、批次信息、稳定性、来源、价格构成。
5. 语言可以刻薄、讽刺，但必须围绕咖啡事实。
6. 不写温和种草，不写“理性讨论一下”，不要把锐评稀释成中立科普。
7. 结尾留一句锋利总结。
